Politique de confidentialité — Inkletter.app

Dernière mise à jour : juin 2026

Qui sommes-nous

Inkletter.app est une application Android de lecture de newsletters conçue pour les liseuses E-Ink. Contact : alex@salemarketingbusiness.com.

Données traitées

  • Adresse e-mail de votre compte Google (identification du compte connecté).
  • Métadonnées et contenu des e-mails identifiés comme newsletters (présence du header List-Unsubscribe), via l'API Gmail en lecture seule (scope gmail.readonly).

Utilisation des données Google (Limited Use)

L'utilisation des informations reçues des API Google par Inkletter.app respecte la Google API Services User Data Policy, y compris les exigences « Limited Use ». Concrètement :

  • Les données Gmail servent uniquement à afficher vos newsletters dans l'application.
  • Aucune donnée n'est transmise à des serveurs tiers : tout le traitement a lieu sur votre appareil.
  • Aucune donnée n'est vendue, partagée, ni utilisée pour de la publicité.
  • Aucun humain ne lit vos données.

Stockage

  • Jetons OAuth : chiffrés sur l'appareil (Android Keystore via Expo SecureStore).
  • Cache de lecture et préférences : stockage local de l'appareil uniquement.
  • Aucune base de données distante, aucun service d'analytique, aucun traceur.

Partage

Aucune donnée n'est partagée avec des tiers. Les seules communications réseau de l'application sont dirigées vers les API officielles de Google (HTTPS uniquement).

Conservation et suppression

Les données restent sur votre appareil. La déconnexion (Réglages → Se déconnecter) ou la désinstallation de l'application supprime les jetons et le cache local. Vous pouvez aussi révoquer l'accès de l'application depuis votre compte Google : myaccount.google.com/permissions.

Modifications

Toute modification de cette politique sera publiée sur cette page avec une nouvelle date de mise à jour.
